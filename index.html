Introduzione a HTML

HTML (HyperText Markup Language) è il linguaggio di markup standard per creare pagine web. Consente di strutturare il contenuto del sito utilizzando vari elementi e tag.

Struttura di base di un documento HTML

<!DOCTYPE html>
<html lang="it">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Titolo della pagina</title>
    </head>
    <body>
        <h1>Benvenuto</h1>
        <p>Questo è un esempio di pagina HTML.</p>
    </body>
</html>

Elementi e Tag HTML

Ogni elemento HTML è racchiuso tra tag. I tag di apertura e chiusura sono scritti tra le parentesi angolari (< e >).

  • Tag di apertura: <tag>
  • Tag di chiusura: </tag>

Esempio di tag:

<p>Questo è un paragrafo.</p>

Tag Comuni

  • <h1> - <h6>: Intestazioni, con <h1> come la più grande e <h6> come la più piccola.
  • <p>: Paragrafo.
  • <a href="url">: Link ipertestuale.
  • <img src="url" alt="descrizione">: Immagine.
  • <ul>: Lista non ordinata.
  • <ol>: Lista ordinata.
  • <li>: Voce di lista.

Attributi HTML

Gli attributi forniscono ulteriori informazioni sugli elementi. Sono scritti all'interno del tag di apertura.

<a href="https://www.example.com">Visita il sito</a>

Commenti HTML

I commenti non vengono visualizzati nel browser e sono utili per annotare il codice.

<!-- Questo è un commento -->

Conclusioni

HTML è un linguaggio fondamentale per la creazione di pagine web. Comprendere la sua sintassi e struttura è essenziale per chiunque desideri sviluppare contenuti online.
